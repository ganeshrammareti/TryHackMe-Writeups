# TryHackMe Writeups

This repository contains my personal notes and walkthroughs while learning cybersecurity on **TryHackMe**.

I created this repository to document my learning process, improve my understanding of different security concepts, and track the rooms I complete during my cybersecurity journey.

The writeups focus on explaining the **concepts, tools, and techniques** used in each room rather than just providing answers.

---

## About Me

I am a cybersecurity student interested in ethical hacking, network security, and Pentesting.
Through these labs, I practice real-world security skills such as:

* Web application security
* Linux fundamentals
* Network enumeration
* Privilege escalation
* Security monitoring
* Basic penetration testing techniques
